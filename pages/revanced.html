Fitur Utama:

• Ikon dan nama aplikasi seperti YouTube bawaan.
• Menonaktifkan pemutaran YouTube Shorts saat meluncurkan YouTube.
• Integrasikan SponsorBlock.
• Menampilkan jumlah tidak suka video menggunakan Return YouTube Dislike API.
• Menampilkan branding premium di layar beranda.
• Menghapus iklan umum.
• Menghapus iklan di pemutar video.
• Menambahkan kontrol gesek volume dan kecerahan.
• Mengaktifkan pemutaran yang diminimalkan dan latar belakang.
• Memungkinkan pengunduhan musik dan video dari YouTube.
Untuk melihat semua fitur bisa langsung ke Revanced Patches

Revanced YouTube yang saya buat ini tidak semua fitur dari Revanced Patches saya aktifkan, karena tidak semua terpakai.
Untuk melihat fitur apa saja yang saya aktifkan bisa ke menu:
klik ikon akun - pengaturan - ReVanced

Cara Memasang:
• Jika masih menggunakan YouTube bawaan dari Play Store, Uninstall update terlebih dahulu, karena YouTube bawaan tidak dapat di hapus cukup dengan Uninstall Pembaharuan saja.
• Kemudian Install terlebih dahulu Microg-nya lalu Install YouTube Revanced-nya.
• Untuk login akun Google, buka Youtube revanced-nya kemudian klik ikon bulat kanan atas.

Catatan:
• Untuk menggunakan fitur unduhan musik dan video silahkan install aplikasi NewPipe terlebih dahulu
Unduh NewPipe
• Untuk menampilkan Short dan postingan komunitas di Beranda atau di tab langganan, matikan 2 menu berikut di pengaturan Revanced:
- Sembunyikan postingan komunitas
 - sembunyikan Short
