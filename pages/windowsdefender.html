Menonaktifkan
Windows Defender

Unduh Aplikasi

• Untuk menonaktifkan Windows Defender silahkan unduh aplikasi Autoruns melalui link dibawah:
Unduh Autoruns
• Setelah Terunduh Ekstrak aplikasinya ke Desktop atau bisa dimanapun asal mundah untuk diakses kembali.

Tahap 1

• Buat restore point terlebih dahulu, untuk berjaga-jaga dikalau ada masalah pada Windows nya
• Kemudian buka Windows Defender lalu matikan Realtime Protection (Wajib!!)
• Jika sudah, Reboot Windows nya ke Safe Mode
• Setelah masuk Windows Safe Mode, buka aplikasi Autoruns tadi dengan "Run As Administrator"
• Klik menu Options - Hilangkan centang "Hide Windows Entries" atau klik logo Windows berwarna biru
• Klik Tab "Services", kemudian pada kotak Quick Filter ketik WinDefend, lalu hilangkan centang disampingnya
• Reboot Windows (keluar dari Safe Mode)

Tahap 2

• Setelah masuk ke normal Windows, tekan tombol Win + R lalu ketik gpedit.msc lalu tekan enter
• Navigasi ke:
"computer configuration - Adminitrative Templates - Windows Component - Microsoft Defender Antivirus"
• Pada bagian kanan klik 2x "Turn off Microsoft Defender Antivirus" lalu pilih Enable kemudian klik OK
• Terakhir jangan pernah buka Windows Defendernya biar tidak jalan kembali
